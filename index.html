Basic Syntax

To link an external stylesheet to your HTML, use the following syntax

<head>
  <link rel="stylesheet" type="text/css" href="yourstylesheet.css">
</head>

When referencing a class, use periods. i.e "."

.selector{
  property1:value;
  property2:value;
}

When referencing an ID, use hashtags. i.e "#"

#selector{
  property1:value;
  property2:value;
}
